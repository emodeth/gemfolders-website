# Gem Folders Website

Welcome to the official repository for the **Gemfolders** website. Gem Folders is "the missing file manager for Gemini," designed to help users organize their Gemini conversations and ideas with native folders, instant search, and bookmarks.

This project is built with **Next.js 16**, **Tailwind CSS 4**, and **TypeScript**, featuring a modern, responsive design with smooth animations powered by **Framer Motion**.

## 🚀 Features

- **Hero Section**: Engaging introduction with a direct call to action.
- **Video Demo**: Showcase of the extension in action.
- **Feature Highlights**: Detailed breakdown of key functionalities (Folders, Search, Bookmarks).
- **Pricing Plans**: Clear overview of available subscriptions.
- **FAQ**: Answers to common user questions.
- **Responsive Design**: Optimized for all devices using Tailwind CSS.
- **Animations**: Smooth transitions and effects using Framer Motion.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Linting**: [ESLint](https://eslint.org/)
